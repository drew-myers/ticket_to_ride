# Project Setup and Cargo Configuration

Initialize the Rust project with proper dependencies and structure.

## Description

Set up the cargo project with all required dependencies:
- clap for CLI parsing
- tokio for async runtime
- reqwest for HTTP/GraphQL calls
- serde/toml for config parsing
- gray_matter for YAML frontmatter
- thiserror/anyhow for error handling

Create the basic directory structure:
```
src/
├── main.rs
├── lib.rs
├── config.rs
├── ticket.rs
├── auth.rs
├── sync.rs
└── github/
    ├── mod.rs
    ├── client.rs
    ├── issues.rs
    ├── projects.rs
    └── subissues.rs
```

## Acceptance Criteria

- [ ] `cargo build` succeeds
- [ ] All dependencies declared in Cargo.toml
- [ ] Basic module structure in place with placeholder code
- [ ] `.gitignore` configured for Rust project
