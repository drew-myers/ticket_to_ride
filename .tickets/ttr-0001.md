# ttr MVP - Core Push Functionality

Epic ticket tracking the MVP implementation of ttr (Ticket To Ride).

The MVP enables the core workflow: create tickets with an AI agent, then run `ttr push` to sync them to GitHub Issues with labels.

## Acceptance Criteria

- [ ] Can run `ttr push` to create GitHub issues from tickets
- [ ] Issues are created with proper title and body
- [ ] Ticket tags are synced as GitHub labels
- [ ] `external-ref: gh-{number}` is written back to ticket files
- [ ] Auth works via GITHUB_TOKEN or `gh auth token`
