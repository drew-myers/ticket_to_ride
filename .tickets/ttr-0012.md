# ttr Phase 3 - GitHub Projects and Relationships

Add GitHub Projects integration and ticket relationship syncing.

## Description

Phase 3 adds the advanced GitHub features:
- Add issues to GitHub Projects
- Set project field values (Type, etc.)
- Sync parent/child as GitHub sub-issues
- Sync dependencies as issue references

## Acceptance Criteria

- [ ] Issues are added to configured GitHub Project
- [ ] Project "Type" field is set based on ticket type
- [ ] Parent tickets create sub-issue relationships
- [ ] Dependencies are shown in issue body as "Depends on #X, #Y"
- [ ] Project schema is validated before sync
