# CLI Implementation

Implement the command-line interface using clap.

## Description

Implement in `src/main.rs`:

### Commands

```
ttr - Sync tickets to GitHub Issues

USAGE:
    ttr <COMMAND>

COMMANDS:
    push      Sync tickets to GitHub Issues
    help      Print help information

PUSH:
    ttr push              # Sync all tickets
    ttr push <ID>...      # Sync specific tickets
```

### Push Command
- Load config from `.tickets/sync.toml`
- Initialize GitHub client with resolved token
- Load tickets (all or specified IDs)
- Run sync engine
- Print results summary

### Output Format
```
Syncing 4 tickets...

CREATE  nw-5c46 → #123  Implement user auth
  └─ Labels: backend, auth

UPDATE  nw-5c40 → #120  Update docs
  └─ Title updated

SKIP    nw-5c35 → #115  (modified outside ttr)

Summary: 1 created, 1 updated, 1 skipped
```

## Acceptance Criteria

- [ ] `ttr push` syncs all tickets
- [ ] `ttr push <id>` syncs specific tickets
- [ ] Clear progress output during sync
- [ ] Summary at end showing created/updated/skipped counts
- [ ] Helpful error messages for common issues
