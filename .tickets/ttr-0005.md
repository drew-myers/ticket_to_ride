# GitHub Authentication

Implement GitHub token resolution for API authentication.

## Description

Resolve GitHub API token from multiple sources in order:

1. `GITHUB_TOKEN` environment variable
2. `gh auth token` command output (GitHub CLI)

Implement in `src/auth.rs`:
- `get_github_token()` function that tries sources in order
- Clear error message if no token found
- Validate token format (should start with `ghp_`, `gho_`, or `github_pat_`)

## Acceptance Criteria

- [ ] Reads GITHUB_TOKEN from environment
- [ ] Falls back to `gh auth token` command
- [ ] Returns clear error when no token available
- [ ] Works when gh CLI is not installed (just tries env var)
